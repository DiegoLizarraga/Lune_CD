# 🎨 Avatar Packs de Lune

Cada subcarpeta aquí es un **pack** de expresiones intercambiable.

```
packs/
└── mi_pack/
    ├── lune_normal.png
    ├── lune_happy.png
    ├── pensando.mp4
    └── ...   (cualquier archivo de lune_face/ que quieras reemplazar)
```

Selecciónalo en **⚙️ Configuración General → 🎨 Modelo visual de Lune**.
Lo que falte en tu pack cae automáticamente al set por defecto (`lune_face/`).

Ver el plan completo (Live2D / VRM estilo Mate-Engine) en `ROADMAP_MODELOS.md`.
